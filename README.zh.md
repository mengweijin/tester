# quickboot

Language: [English](README.md)

## 介绍
基于Cucumber的BDD自动化接口测试平台

支持配置数据源，配置接口，通过Excel导入/导出测试用例。在线执行测试用例，查看报告。测试用例重复执行等。

Behaviour-Driven Development(BDD) Test Automation
